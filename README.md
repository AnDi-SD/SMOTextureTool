# English

## 📌 Description

**SMO Texture Tool** is a utility for working with `.smo` model files from the **Winx Club PC game**.

It allows you to:
- Extract textures from models
- Preview textures in a graphical interface
- Replace textures with custom ones
- Repack files with updated content

---

## 📥 Download

👉 Download latest version:  
https://github.com/AnDi-SD/SMOTextureTool/releases

---

## 🚀 How to use

### 1. Open file
- Click **"Выбрать"**
- Select `.smo` file

### 2. Extract textures
- Click **"Извлечь все текстуры"**
- Choose output folder

### 3. Replace textures
- Click **"Папка с заменами"**
- Select folder with edited textures  
(Filenames must match extracted ones)

### 4. Save new file
- Click **"Сохранить новый SMO"**
- Save modified file

---

## ⚠️ Notes

- Repacking currently supports square textures only
- Some models may use grayscale textures with color applied in-game

# Русский

## 📌 Описание

**SMO Texture Tool** — инструмент для работы с `.smo` файлами моделей из игры **Winx Club для ПК**.

Позволяет:
- Извлекать текстуры
- Просматривать их в графическом интерфейсе
- Заменять на свои
- Пересобирать файл

---

## 📥 Скачать

👉 Скачать последнюю версию:  
https://github.com/AnDi-SD/SMOTextureTool/releases

---

## 🚀 Как использовать

### 1. Открыть файл
- Нажать **"Выбрать"**
- Выбрать `.smo`

### 2. Извлечь текстуры
- Нажать **"Извлечь все текстуры"**
- Указать папку

### 3. Заменить текстуры
- Нажать **"Папка с заменами"**
- Выбрать папку с изменёнными текстурами  
(имена должны совпадать)

### 4. Сохранить файл
- Нажать **"Сохранить новый SMO"**
- Сохранить

---

## ⚠️ Важно

- Замена пока поддерживает только квадратные текстуры
- Некоторые модели используют серые текстуры с цветом, применяемым в игре
